# Story 7.4 — Materiais: CursoPage seção "Materiais do Curso"

## Status

Ready

## Story

**Como** aluno,
**Quero** ver os materiais relacionados ao curso diretamente na página do curso,
**Para que** eu acesse os arquivos de apoio sem precisar sair da página e ir até Materiais.

## Contexto

Última story do Epic 7. Adiciona a seção "Materiais do Curso" à `CursoPage` (criada no Epic 6, Story 6.2). A seção aparece entre "Conteúdo do Curso" e o fim da página, e só é renderizada se houver materiais vinculados ao curso. O download usa o mesmo padrão de presigned URL da `MateriaisPage`.

## Acceptance Criteria

### AC1 — Query de materiais
- [ ] `CursoPage` faz fetch separado dos materiais do curso após carregar o módulo:
  ```javascript
  supabase
    .from('materials')
    .select('id, title, description, file_size')
    .eq('module_id', moduleId)
    .order('created_at', { ascending: false })
  ```
- [ ] Materiais carregados em estado separado (`materials`, `loadingMaterials`)

### AC2 — Renderização da seção
- [ ] Seção com título "Materiais do Curso" (mesmo estilo de h2 das seções existentes: `border-bottom`, Space Grotesk bold)
- [ ] Seção **oculta** se não houver materiais vinculados ao curso
- [ ] Posicionada após a seção "Conteúdo do Curso"
- [ ] Cada item exibe: ícone FILE, título, descrição (se houver), tamanho formatado, botão "Download"

### AC3 — Download
- [ ] Botão "Download" chama `/api/materials/:id/download` com Bearer token (mesmo padrão de `MateriaisPage`)
- [ ] Estado de loading por item: botão muda para "Gerando link..." enquanto aguarda
- [ ] Link abre em nova aba (`window.open(url, '_blank')`)

### AC4 — UX e estilo
- [ ] Inline styles com `var(--token)` — sem CSS externo
- [ ] Hover state no item (background sutil, consistent com resto da página)
- [ ] Loading de materiais não bloqueia o restante da página (fetch independente)

### AC5 — Sem regressões
- [ ] Seções existentes da CursoPage inalteradas (breadcrumb, título, subtítulo, botão iniciar, sobre o curso, conteúdo do curso)
- [ ] `npm test` passa (32/32)

## Escopo Técnico

### Arquivos a Modificar
- `src/plataforma/pages/CursoPage.jsx` — adicionar estado de materiais + fetch + seção renderizada

### NÃO fazer nesta story
- Modificar `MateriaisPage` (feito na 7.3)
- Mostrar materiais de "Conteúdo Extra" na CursoPage
- Criar componente compartilhado MaterialItem (sem abstração prematura)

### Padrões
- `import { supabase } from '../../lib/supabase'` (já importado em CursoPage)
- `useAuth` para acessar `session.access_token` no download
- Fetch de materiais no mesmo `useEffect` que carrega o módulo (ou separado com mesmo `moduleId` dep)

### Referência de estilo de seção (consistente com CursoPage existente)
```jsx
<h2 style={{ fontSize: '1.15rem', fontWeight: 700, margin: '0 0 12px 0', paddingBottom: '10px', borderBottom: '1px solid var(--line)' }}>
  Materiais do Curso
</h2>
```

## Tarefas de Implementação

- [ ] Renderizar seção "Materiais do Curso" condicionalmente (oculta se vazia)

- [ ] Seções existentes da CursoPage intactas
- [ ] `npm test` passa
