# Story 7.3 — Materiais: MateriaisPage agrupada por curso

## Status

Ready

## Story

**Como** aluno,
**Quero** ver os materiais organizados por curso na página de Materiais,
**Para que** eu encontre rapidamente os arquivos relevantes ao curso que estou estudando.

## Contexto

Continuação da Story 7.2 (admin já associa materiais a cursos). Esta story refatora `MateriaisPage` para exibir os materiais em seções por curso, seguidas de uma seção "Conteúdo Extra" para materiais sem curso associado.

O agrupamento é feito no cliente (JS), após um único fetch que retorna todos os materiais com o nome do curso via join. A ordem das seções segue o `order` dos cursos definido pelo admin.

## Acceptance Criteria

### AC1 — Query e agrupamento
- [ ] Query: `supabase.from('materials').select('id, title, description, file_size, created_at, module_id, modules(id, title, order)').order('created_at', { ascending: false })`
- [ ] Materiais agrupados no cliente por `module_id`:
  - Um grupo por curso (identificado por `modules.id` + `modules.title`)
  - Um grupo "Conteúdo Extra" para `module_id IS NULL`
- [ ] Grupos de cursos ordenados por `modules.order ASC`
- [ ] Grupo "Conteúdo Extra" sempre ao final

### AC2 — Renderização
- [ ] Cada seção exibe: título do curso como cabeçalho de seção (ou "Conteúdo Extra")
- [ ] Cada seção lista os materiais do grupo com o mesmo layout atual (ícone, título, descrição, tamanho, botão Download)
- [ ] Se um grupo tiver 0 materiais: não renderizar a seção (grupos vazios ocultos)
- [ ] Se não houver materiais em nenhum grupo: mensagem "Nenhum material disponível ainda."

### AC3 — Cabeçalho de seção
- [ ] Título do curso em destaque (font Space Mono uppercase, cor accent, separador visual)
- [ ] Subtítulo com contagem: "X material(is)"
- [ ] Separador `border-bottom` ou margem entre seções

### AC4 — Download
- [ ] Lógica de download mantida intacta (`/api/materials/:id/download` com presigned URL)
- [ ] Estado de loading por item preservado

### AC5 — Sem regressões
- [ ] Layout geral (sidebar, header da página) inalterado
- [ ] `npm test` passa (32/32)

### Lógica de agrupamento (referência)
```javascript
// Após fetch de materials (cada item tem modules: { id, title, order } ou null)
const grouped = {};
const extra = [];

for (const m of materials) {
  if (!m.module_id) { extra.push(m); continue; }
  const key = m.module_id;
  if (!grouped[key]) grouped[key] = { title: m.modules.title, order: m.modules.order, items: [] };
  grouped[key].items.push(m);
}

const sections = Object.values(grouped).sort((a, b) => a.order - b.order);
if (extra.length > 0) sections.push({ title: 'Conteúdo Extra', order: Infinity, items: extra });
```
