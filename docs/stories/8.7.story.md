# Story 8.7 — Blog: Comments — Sistema de comentários

## Status

Ready

## Story

**Como** aluno autenticado,
**Quero** comentar nos artigos do blog,
**Para que** eu interaja com o conteúdo e com outros membros.

**Como** admin,
**Quero** moderar os comentários (aprovar, rejeitar),
**Para que** o blog mantenha qualidade e segurança.

## Contexto

Última story do Epic 8. Adiciona seção de comentários ao final de `BlogPostPage` e `PlataformaBlogPostPage`. Comentários ficam em `status=pending` até aprovação do admin. Admin vê e modera comentários pendentes em uma seção dedicada dentro de `AdminBlogPage`.

## Acceptance Criteria

### AC1 — Comentários nas páginas de artigo
- [ ] Seção "Comentários" ao final de `BlogPostPage` e `PlataformaBlogPostPage`
- [ ] Exibe apenas comentários `status='approved'`
- [ ] Cada comentário exibe: nome do autor, data, texto
- [ ] Ordenados por `created_at ASC`
- [ ] Seção "Deixe um comentário" visível apenas para usuários autenticados
- [ ] Para não-autenticados: mensagem "Faça login para comentar." com link para `/plataforma/login`

### AC2 — Formulário de comentário
- [ ] Textarea para o comentário (obrigatório, mín. 10 chars)
- [ ] Botão "Enviar comentário"
- [ ] INSERT em `post_comments`: `{ post_id, user_id: user.id, user_name: profile.full_name ?? user.email, content, status: 'pending' }`
- [ ] Após envio: toast "Comentário enviado! Aguardando moderação." + limpa textarea
- [ ] Estado de loading no botão durante envio

### AC3 — Moderação no admin
- [ ] Nova aba ou seção "Comentários Pendentes" em `AdminBlogPage`
- [ ] Lista comentários com `status='pending'` com: post title, autor, data, texto
- [ ] Botão "Aprovar" → `update status='approved'`
- [ ] Botão "Rejeitar" → `update status='rejected'`
- [ ] Badge com contagem de pendentes no título da seção
- [ ] Após ação: item some da lista (feedback imediato)

### AC4 — Sem regressões
- [ ] BlogPostPage e PlataformaBlogPostPage com conteúdo inalterado acima da seção de comentários
- [ ] `npm test` passa (32/32)

## Escopo Técnico

### Arquivos a Modificar
- `src/pages/BlogPostPage.jsx` — adicionar seção de comentários
- `src/plataforma/pages/PlataformaBlogPostPage.jsx` — adicionar seção de comentários
- `src/plataforma/pages/admin/AdminBlogPage.jsx` — adicionar aba/seção de moderação

### Padrões
- Inline styles com `var(--token)`
- `import { useAuth } from '../context/AuthContext'` para verificar autenticação
- `supabase.from('post_comments').insert(...)` com JWT do usuário (RLS valida autenticação)
- Toast via `sonner`

### NÃO fazer nesta story
- Notificações por e-mail ao admin sobre novos comentários
- Edição de comentário pelo autor
- Replies/threads de comentários
