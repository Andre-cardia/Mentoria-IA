# Story 9.1 — RecentPostsWidget

## Status

InProgress

## Story

**Como** visitante do blog,
**Quero** ver uma lista de posts recentes na sidebar,
**Para que** eu descubra outros artigos e navegue mais pelo blog.

## Contexto

Componente reutilizavel que faz query independente ao Supabase para buscar os 5 posts mais recentes. Sera usado dentro do BlogSidebar em BlogPage e BlogPostPage. Design segue o design system existente (CSS variables do `index.html` e `plataforma.html`).

## Acceptance Criteria

### AC1 — Componente RecentPostsWidget
- [x] Criar `src/components/blog/RecentPostsWidget.jsx`
- [x] Query Supabase: `posts` where `status=published`, `visibility=public`, `published_at <= now()`, order `published_at desc`, limit 5
- [x] Exibir lista com: titulo (link para `/blog/{slug}`), data formatada (pt-BR)
- [x] Titulo truncado com `text-overflow: ellipsis` se muito longo (max 2 linhas)
- [x] Heading "Posts Recentes" com estilo `Space Grotesk, .9rem, var(--text)`
- [x] Items com `font-size: .85rem`, cor `var(--muted)` na data

### AC2 — Estados
- [x] Loading: placeholder sutil (3 linhas skeleton ou texto "Carregando...")
- [x] Vazio: nao renderizar o widget (retorna null)
- [x] Erro: nao renderizar (graceful fallback, sem crash)

### AC3 — Exclusao do post atual
- [x] Prop opcional `excludePostId` para excluir o post sendo lido (evitar duplicata na sidebar do BlogPostPage)

## Scope

### IN
- Componente `RecentPostsWidget.jsx`
- Query independente ao Supabase

### OUT
- Integracao na sidebar (Story 9.3)
- Layout 2 colunas (Story 9.3/9.4)
