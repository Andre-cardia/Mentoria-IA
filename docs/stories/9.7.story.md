# Story 9.7 — InlinePromo banner horizontal

## Status

InProgress

## Story

**Como** dono do produto,
**Quero** um banner promocional dentro do corpo do artigo,
**Para que** leitores engajados vejam a oferta no momento de maior atencao.

## Contexto

Banner horizontal inserido entre o conteudo do artigo e os share buttons. Promove a Mentoria como CTA principal. Visual distinto do conteudo para nao confundir com parte do artigo.

## Acceptance Criteria

### AC1 — Componente InlinePromo
- [x] Criar `src/components/blog/InlinePromo.jsx`
- [x] Props: `title`, `description`, `cta`, `href`, `badge` (opcional)
- [x] Layout horizontal: texto a esquerda, botao CTA a direita
- [x] Background: `var(--panel-2)` com borda accent left (`border-left: 3px solid var(--accent)`)
- [x] Padding `20px 24px`, border-radius `8px`
- [x] Tipografia: titulo `Space Grotesk, 1rem, 600`, descricao `.9rem, var(--muted)`

### AC2 — Integracao no BlogPostPage
- [x] Inserir `<InlinePromo />` entre o conteudo e os ShareButtons em `BlogPostPage.jsx`
- [x] Dados: `title="Mentoria Zero-to-Hero IA"`, `description="Aprenda a construir com IA de verdade. 2 encontros por semana."`, `cta="Ver planos"`, `href="/#planos"`, `badge="A partir de R$ 497/mes"`
- [x] Margem: `margin: 32px 0`

### AC3 — Integracao no PlataformaBlogPostPage
- [x] Inserir mesmo `<InlinePromo />` em `PlataformaBlogPostPage.jsx`
- [x] Mesma posicao (entre conteudo e ShareButtons)

### AC4 — Responsividade
- [x] Mobile (<768px): layout vertical (texto em cima, botao embaixo full width)

## Scope
