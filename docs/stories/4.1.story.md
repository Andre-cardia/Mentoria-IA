# Story 4.1 — Página Inicial — Dashboard do Aluno

## Status

Ready

## Story

**Como** aluno da plataforma Mentoria IA,
**Quero** ver uma página inicial personalizada ao entrar na plataforma,
**Para** acompanhar meu progresso, descobrir novas aulas e me manter motivado a estudar.

## Critérios de Aceite

- [ ] AC-1: Rota `/inicio` exibe o Dashboard; redirect padrão `*` passa a apontar para `/inicio`
- [ ] AC-2: Saudação dinâmica baseada no horário: "Bom dia" (05h–11h59), "Boa tarde" (12h–17h59), "Boa noite" (18h–04h59) + nome do aluno (de `profiles.full_name`)
- [ ] AC-3: Barra de progresso exibe `aulas concluídas / total de aulas` com percentual (usa dados do `lesson_progress` e `lessons`)
- [ ] AC-4: Cards de stats à direita da saudação: **Aulas assistidas** (count de `lesson_progress`), **Certificados** (0 — placeholder para Epic futuro), **Dias estudando** (count de datas distintas em `lesson_progress.completed_at`)
- [ ] AC-5: Seção "Últimos Lançamentos" lista as 4 aulas mais recentes (order by `lessons.created_at DESC`), com título, módulo e tipo
- [ ] AC-6: Seção "Continue Assistindo" exibe a primeira aula não concluída do primeiro módulo em ordem; se todas concluídas exibe mensagem de parabéns
- [ ] AC-7: Seção "Ranking da Semana" exibe top 5 alunos com mais aulas concluídas nos últimos 7 dias (via RPC Supabase), com posição, nome e pontos; o aluno atual é destacado

- [ ] AC-9: "Início" aparece como primeiro item na navegação do sidebar (`Layout.jsx`)

| `profiles.full_name` nulo para usuários antigos | Média | Baixo | Fallback para `user.email.split('@')[0]` |
| Muitas queries paralelas na montagem do dashboard | Média | Baixo | `Promise.all` agrupa queries; loading state unificado |

### RPC get_weekly_ranking
```sql
CREATE OR REPLACE FUNCTION public.get_weekly_ranking()
RETURNS TABLE(user_name text, points bigint, is_current_user boolean)
LANGUAGE sql SECURITY DEFINER AS $$
  SELECT
    COALESCE(p.full_name, split_part(u.email, '@', 1)) AS user_name,
    COUNT(lp.id)::bigint AS points,
    (lp.user_id = auth.uid()) AS is_current_user
  FROM lesson_progress lp
  JOIN auth.users u ON u.id = lp.user_id
  LEFT JOIN profiles p ON p.user_id = lp.user_id
  WHERE lp.completed_at >= NOW() - INTERVAL '7 days'
  GROUP BY lp.user_id, p.full_name, u.email
  ORDER BY points DESC
  LIMIT 10;
$$;
```

### Continue Assistindo — lógica
```js
// Primeira aula (em ordem de módulo/aula) não presente em completedIds
const nextLesson = modules
  .flatMap(m => (m.lessons || []).map(l => ({ ...l, moduleId: m.id })))
  .sort((a, b) => a.order - b.order) // aulas por order dentro do módulo
  .find(l => !completedIds.has(l.id));
```
