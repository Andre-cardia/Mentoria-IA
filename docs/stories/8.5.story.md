# Story 8.5 — Blog: Public — Artigo individual + SEO + social share

## Status

Ready

## Story

**Como** visitante,
**Quero** ler um artigo completo em `/blog/:slug` com SEO configurado e botões para compartilhar,
**Para que** eu consuma o conteúdo e possa indicar para outras pessoas.

## Contexto

Cria a `BlogPostPage` na landing page. Renderiza o conteúdo Tiptap JSON como HTML. Injeta meta tags de SEO/Open Graph via `react-helmet-async`. Exibe botões de compartilhamento para WhatsApp, Twitter/X e LinkedIn.

**Limitação conhecida:** `react-helmet-async` é client-side. Meta tags OG são visíveis para WhatsApp e Telegram (executam JS) mas não para crawlers do Twitter/X e LinkedIn. Limitação aceita nesta fase.

## Acceptance Criteria

### AC1 — Fetch e renderização
- [ ] Query por slug: `supabase.from('posts').select('id, title, content_json, cover_url, author_name, published_at, seo_title, seo_description, visibility, status, post_tags(tags(name, slug))').eq('slug', slug).single()`
- [ ] Valida: `status IN ('published','scheduled') AND published_at <= NOW()` — 404 se não atender
- [ ] Conteúdo Tiptap JSON renderizado via `generateHTML` de `@tiptap/html` ou via `EditorContent` com `editable=false`
- [ ] Cover image exibida como banner de cabeçalho (se existir)

### AC2 — Layout do artigo
- [ ] Breadcrumb: `Blog › [Título]` com link para `/blog`
- [ ] Cover image como hero (largura total, height fixo com `object-fit: cover`)
- [ ] Título h1, autor, data formatada, badges de tags
- [ ] Corpo do artigo com tipografia legível (line-height 1.8, font-size 1.05rem)
- [ ] Estilos inline para elementos gerados pelo Tiptap (h2, h3, ul, ol, blockquote, img, a)

### AC3 — SEO / Open Graph
- [ ] `react-helmet-async` instalado e `HelmetProvider` adicionado ao entry point `src/main.jsx`
- [ ] `<title>` = `seo_title ?? title`
- [ ] `<meta name="description">` = `seo_description`
- [ ] `<meta property="og:title">` = `seo_title ?? title`
- [ ] `<meta property="og:description">` = `seo_description`
- [ ] `<meta property="og:image">` = `cover_url` (se existir)
- [ ] `<meta property="og:url">` = URL atual
- [ ] `<meta property="og:type" content="article">`

### AC4 — Compartilhamento social
- [ ] Botão WhatsApp: `https://wa.me/?text=<title> <url>` (abre em nova aba)
- [ ] Botão Twitter/X: `https://twitter.com/intent/tweet?text=<title>&url=<url>`
- [ ] Botão LinkedIn: `https://www.linkedin.com/sharing/share-offsite/?url=<url>`
- [ ] Botões agrupados abaixo do conteúdo, antes dos comentários

### AC5 — Estados
- [ ] Loading: "Carregando artigo..."
- [ ] Post não encontrado / não publicado: "Artigo não encontrado." com link de volta para `/blog`

### AC6 — Rota
- [ ] Rota `/blog/:slug` adicionada em `src/main.jsx` com lazy import

### AC7 — Sem regressões
- [ ] `/blog` (listagem) inalterada
- [ ] `npm test` passa (32/32)
