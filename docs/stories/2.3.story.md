# Story 2.3 — Duplicar Aula

## Status

Ready for Review

## Story

**Como** administrador da plataforma,
**Quero** duplicar uma aula existente com um clique,
**Para que** eu possa criar aulas similares rapidamente sem redigitar título, URL e duração.

## Contexto

**Pré-requisito:** Story 2.1 Done (para ter o padrão de toast e modal estabelecido).

Arquivo alvo: [AdminModulosPage.jsx](../../src/plataforma/pages/admin/AdminModulosPage.jsx)

**Motivação:** Ao criar módulos com muitas aulas similares (ex: exercícios práticos seguindo o mesmo template), o admin pode usar uma aula como base, duplicar e ajustar apenas o que muda.

### AC1 — Botão Duplicar na Linha da Aula

- [x] O botão fica visível na linha, com label "Duplicar"

### Algoritmo

```jsx
async function duplicateLesson(lesson, moduleId) {
  // 1. Incrementar order de todas as aulas após a original
  const affectedLessons = lessons[moduleId].filter(l => l.order > lesson.order);
  await Promise.all(
    affectedLessons.map(l =>
      supabase.from('lessons').update({ order: l.order + 1 }).eq('id', l.id)
    )
  );

  // 2. Inserir a cópia
  const { data, error } = await supabase.from('lessons').insert({
    title: `${lesson.title} (cópia)`,
    video_url: lesson.video_url,
    duration: lesson.duration,
    module_id: moduleId,
    order: lesson.order + 1,
  }).select().single();

  if (error) { toast.error('Erro ao duplicar aula'); return; }

  // 3. Atualizar estado local
  const updated = [...lessons[moduleId]];
  updated.splice(lesson.order + 1, 0, data); // inserir na posição correta
  setLessons(p => ({ ...p, [moduleId]: updated.map((l, i) => ({ ...l, order: i })) }));
  toast.success('Aula duplicada com sucesso');
}
```
