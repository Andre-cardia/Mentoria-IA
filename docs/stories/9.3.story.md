# Story 9.3 — BlogSidebar + BlogPage layout 2 colunas

## Status

InProgress

## Story

**Como** visitante do blog,
**Quero** ver uma sidebar com posts recentes e banners ao lado do grid de artigos,
**Para que** eu descubra mais conteudo e conheca os produtos oferecidos.

## Contexto

Refatoracao do layout de `BlogPage.jsx` de grid full-width para layout 2 colunas com sidebar. A sidebar agrupa `RecentPostsWidget` (9.1) e `PromoWidget` (9.2). Mobile (<768px) a sidebar colapsa abaixo do conteudo principal.

## Acceptance Criteria

### AC1 — BlogSidebar container
- [x] Criar `src/components/blog/BlogSidebar.jsx`
- [x] Props: `children` (slot pattern) ou composicao interna fixa
- [x] Renderiza: RecentPostsWidget + PromoWidget (Pulsemind imagem) + PromoWidget (Pulsemind video) + PromoWidget (Mentoria texto)
- [x] Prop opcional `excludePostId` para repassar ao RecentPostsWidget

### AC2 — BlogPage layout 2 colunas
- [x] Refatorar `src/pages/BlogPage.jsx`
- [x] Layout: `display: grid; grid-template-columns: 1fr 300px; gap: 32px`
- [x] Coluna esquerda: tag filters + grid de posts (manter `repeat(auto-fill, minmax(280px, 1fr))` dentro da coluna)
- [x] Coluna direita: BlogSidebar
- [x] `maxWidth: 1200px` mantido no container

### AC3 — Dados dos PromoWidgets
- [x] Pulsemind (imagem): `image=pulsemind01.jpeg`, `title="Pulsemind"`, `description="IA generativa para empresas e equipes de marketing"`, `cta="Conhecer"`, `href="https://pulsemind.com.br"`
- [x] Pulsemind (video): `video=pulsemind-promo.mp4`, `title="Pulsemind em acao"`, `cta="Conhecer"`, `href="https://pulsemind.com.br"`
- [x] Mentoria (texto): `badge="A partir de R$ 497/mes"`, `title="Mentoria Zero-to-Hero IA"`, `description="2 encontros por semana. Pare de estudar como espectador."`, `cta="Ver planos"`, `href="/#planos"`

### AC4 — Responsividade mobile
- [x] `@media (max-width: 768px)`: grid muda para `grid-template-columns: 1fr`
- [x] Sidebar renderiza abaixo do grid de posts
- [x] Nenhum conteudo cortado ou com overflow horizontal
