# Story 9.6 — FeaturedPost hero

## Status

InProgress

## Story

**Como** visitante do blog,
**Quero** ver o post em destaque em formato hero no topo,
**Para que** o conteudo principal me chame atencao imediatamente.

## Contexto

Banner hero full-width (dentro do container 1200px) que exibe o post marcado como `featured=true`. Posicionado entre a hero section ("Blog" + subtitulo) e os tag filters. Se nenhum post for featured, nao renderiza nada.

## Acceptance Criteria

### AC1 — Componente FeaturedPost
- [x] Criar `src/components/blog/FeaturedPost.jsx`
- [x] Query Supabase: `posts` where `featured=true`, `status=published`, `visibility=public`, limit 1
- [x] Exibir: cover image (full width, 320px height), titulo, `seo_description` como excerpt, autor, data
- [x] Botao CTA "Ler artigo" que navega para `/blog/{slug}`
- [x] Layout: imagem a esquerda (60%), texto a direita (40%) — ou imagem full com overlay de texto
- [x] Tipografia: titulo `clamp(1.4rem, 3vw, 2rem)`, excerpt `var(--muted)`

### AC2 — Estados
- [x] Sem featured: retorna `null` (nao renderiza nada)
- [x] Loading: skeleton ou nao renderizar ate carregar
- [x] Fallback sem cover: background `var(--panel-2)` com titulo centralizado

### AC3 — Integracao na BlogPage
- [x] Inserir `<FeaturedPost />` entre hero section e tag filters em `BlogPage.jsx`
- [x] Full width dentro do container `maxWidth: 1200px`
- [x] Nao duplicar o featured post no grid de posts abaixo (excluir via `id`)

### AC4 — Responsividade
- [x] Mobile (<768px): layout vertical (imagem em cima, texto embaixo)
- [x] Imagem reduz para 200px height no mobile

## Scope

### IN
- Componente `FeaturedPost.jsx`
- Integracao na `BlogPage.jsx`
- Exclusao do post featured do grid

### OUT
- Integracao na PlataformaBlogPage (pode ser feito depois)
