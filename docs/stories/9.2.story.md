# Story 9.2 — PromoWidget (3 variantes)

## Status

InProgress

## Story

**Como** dono do produto,
**Quero** banners promocionais configuráveis na sidebar do blog,
**Para que** eu promova a Mentoria e o Pulsemind para os visitantes.

## Contexto

Banner promocional com 3 variantes (imagem, video, texto) controlado via props. Usado na sidebar do blog para promover produtos. Assets do Pulsemind ja existem em `src/assets/`.

## Acceptance Criteria

### AC1 — Variante image
- [x] Criar `src/components/blog/PromoWidget.jsx`
- [x] Props: `variant="image"`, `image`, `title`, `description`, `cta`, `href`
- [x] Renderiza imagem no topo, titulo, descricao e botao CTA
- [x] Imagem com `border-radius: 6px`, `object-fit: cover`
- [x] Botao CTA com estilo accent (`background: var(--accent)`, `color: #000`)
- [x] Link abre em nova aba (`target="_blank"`)

### AC2 — Variante video
- [x] Props: `variant="video"`, `video`, `title`, `cta`, `href`
- [x] Video com `autoPlay`, `muted`, `loop`, `playsInline`
- [x] Sem controles nativos (`controls={false}`)
- [x] Botao flutuante mute/unmute no canto inferior direito do video
- [x] Icone: `🔇` quando mutado, `🔊` quando com audio
- [x] Botao com fundo semi-transparente (`rgba(0,0,0,.6)`)
- [x] Titulo e CTA abaixo do video

### AC3 — Variante text
- [x] Props: `variant="text"`, `badge`, `title`, `description`, `cta`, `href`
- [x] Badge no topo (pill com `var(--accent-soft)` e `var(--accent)`)
- [x] Titulo grande, descricao, botao CTA
- [x] Background `var(--panel-2)`, borda `var(--line)`

### AC4 — Container comum
- [x] Todas as variantes dentro de um container com `border-radius: 8px`, `overflow: hidden`
- [x] Borda `1px solid var(--line)`
- [x] Padding interno `16px` para texto/CTA (nao para imagem/video que sangram)
- [x] `margin-bottom: 20px` para espaçamento entre widgets

## Scope

### IN
- Componente `PromoWidget.jsx` com 3 variantes
- Mute/unmute toggle no video

### OUT
- Integracao na sidebar (Story 9.3)
- Admin para gerenciar banners (futuro)
