# Story 9.4 — BlogPostPage layout com sidebar sticky

## Status

InProgress

## Story

**Como** leitor de um artigo no blog,
**Quero** ver posts recentes e banners promocionais na sidebar enquanto leio,
**Para que** eu descubra outros conteudos sem sair da pagina.

## Contexto

Refatoracao do layout de `BlogPostPage.jsx` de coluna unica (740px) para layout 2 colunas com sidebar sticky. A sidebar acompanha o scroll do leitor. Mobile (<768px) colapsa abaixo do artigo.

## Acceptance Criteria

### AC1 — Layout 2 colunas
- [x] Refatorar `src/pages/BlogPostPage.jsx`
- [x] Container: `max-width: 1060px` (740 artigo + 40 gap + 280 sidebar)
- [x] Layout: `display: grid; grid-template-columns: 1fr 280px; gap: 40px`
- [x] Coluna esquerda: conteudo do artigo (tudo que existe hoje)
- [x] Coluna direita: BlogSidebar com `position: sticky; top: 80px` (abaixo do header)

### AC2 — Sidebar com exclusao do post atual
- [x] Passar `excludePostId={post.id}` para BlogSidebar/RecentPostsWidget
- [x] O post sendo lido NAO aparece na lista de recentes

### AC3 — Responsividade mobile
- [x] `@media (max-width: 768px)`: `grid-template-columns: 1fr`
- [x] Sidebar renderiza abaixo dos comentarios
- [x] Sticky desativado no mobile

### AC4 — PlataformaBlogPostPage
- [x] Aplicar mesmo layout 2 colunas em `src/plataforma/pages/PlataformaBlogPostPage.jsx`
- [x] Sidebar dentro do `<Layout>` existente (respeitar sidebar da plataforma)
- [x] Se espaco insuficiente (plataforma tem sidebar 240px + conteudo), sidebar do blog colapsa via media query
