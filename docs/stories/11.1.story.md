# Story 11.1 — Pagina Solicitar Proposta

## Status

Done

## Story

**Como** potencial cliente corporativo,
**Quero** acessar uma pagina dedicada para solicitar uma proposta da Neural Hub,
**Para que** eu entenda o valor da solucao antes de preencher o formulario.

## Contexto

Os botoes CTA "Solicitar Proposta" da pagina Neural Hub precisavam de um destino real. A pagina deve convencer o visitante com conteudo comercial, seguir o design system da marca e registrar dados para acompanhamento posterior.

## Planejamento

1. Criar rota publica `/solicitar-proposta`.
2. Construir hero com mensagem comercial, indicadores e imagem de treinamento corporativo de IA.
3. Adicionar secoes de diagnostico, metodologia, entregaveis e formulario.
4. Persistir dados em `proposal_requests` usando Supabase.
5. Atualizar CTAs da landing page para apontar para a nova rota.
