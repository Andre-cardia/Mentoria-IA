# Story 8.4 — Blog: Public — Listagem de artigos

## Status

Ready

## Story

**Como** visitante,
**Quero** acessar `/blog` e ver a lista de artigos públicos publicados,
**Para que** eu consuma o conteúdo sem precisar criar conta.

## Contexto

Cria a `BlogPage` na landing page (entry point `src/main.jsx`). Exibe cards dos posts públicos publicados, ordenados por data. Inclui filtro por tag. A página precisa ser criada como uma nova rota na landing, não na plataforma.

## Acceptance Criteria

### AC1 — Query
- [ ] `supabase.from('posts').select('id, title, slug, cover_url, seo_description, published_at, author_name, post_tags(tags(name, slug))').eq('visibility', 'public').eq('status', 'published').lte('published_at', new Date().toISOString()).order('published_at', { ascending: false })`
- [ ] RLS pública permite leitura sem autenticação

### AC2 — Listagem de cards
- [ ] Cada card exibe: cover (ou placeholder), título, descrição (seo_description), autor, data formatada (ex: "13 abr 2026"), tags como badges
- [ ] Click no card navega para `/blog/:slug`
- [ ] Hover: elevação sutil (border-color transition)

### AC3 — Filtro por tag
- [ ] Linha de badges de tags com todas as tags presentes nos posts listados
- [ ] Click em uma tag filtra a listagem client-side
- [ ] Badge "Todos" reseta o filtro
- [ ] Tag ativa com destaque visual (accent color)

### AC4 — Estados
- [ ] Loading: "Carregando artigos..."
- [ ] Vazio: "Nenhum artigo publicado ainda."
- [ ] Cover ausente: placeholder com cor de fundo `var(--panel-2)`

### AC5 — Roteamento na landing
- [ ] `BlogPage` importada como lazy em `src/main.jsx`
- [ ] Rota `/blog` adicionada ao React Router em `src/main.jsx`
- [ ] Link "Blog" adicionado no header/nav da landing page existente

### AC6 — Sem regressões
- [ ] Landing page existente inalterada
- [ ] `npm test` passa (32/32)

## Escopo Técnico

### Arquivos a Criar
- `src/pages/BlogPage.jsx` (ou `src/blog/BlogPage.jsx` — seguir estrutura existente da landing)

### Arquivos a Modificar
- `src/main.jsx` — adicionar rota `/blog` e lazy import
- Header/nav da landing — adicionar link "Blog" (verificar componente existente)

### Padrões
- Inline styles com `var(--token)` — sem CSS externo
- `import { createClient } from '@supabase/supabase-js'` ou importar do `lib/supabase` se acessível
- `import { lazy, Suspense } from 'react'` no main.jsx

### NÃO fazer nesta story
- Página individual do post (Story 8.5)
- Posts privados na plataforma (Story 8.6)
- Comentários (Story 8.7)
