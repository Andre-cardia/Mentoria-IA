# Story 6.2 — Curso: CursoPage + CursosPage + rotas

## Status

Ready

## Story

**Como** aluno autenticado,
**Quero** ver uma página dedicada para cada curso com título, subtítulo, descrição e lista de aulas, e poder iniciar o curso com um clique,
**Para que** eu entenda o que vou aprender antes de começar e acesse o conteúdo de forma direta.

## Contexto

Continuação da Story 6.1 (subtitle já existe no banco e admin atualizado). Esta story cria a experiência de navegação completa do aluno:

1. **`CursosPage`** — substitui o accordion da `ModulosPage` por cards clicáveis
2. **`CursoPage`** — nova página de detalhes do curso (`/cursos/:moduleId`)
3. **Rotas** — `/cursos` e `/cursos/:moduleId/aulas/:lessonId` substituem as antigas
4. **Breadcrumbs** — `CursoPage` e `LessonPage` atualizados
5. **Redirecionamentos** — `/modulos` e `/modulos/:id/aulas/:id` redirecionam para as novas rotas

## Acceptance Criteria

### AC1 — CursosPage (`/cursos`)
- [ ] Página renderiza lista de cursos como **cards** (não accordion)
- [ ] Cada card exibe: título do curso, subtítulo (se preenchido), badge com quantidade de aulas, barra de progresso do aluno naquele curso
- [ ] Clicar no card navega para `/cursos/:moduleId`
- [ ] Título da página: "Cursos"
- [ ] Breadcrumb label na página: sem breadcrumb (é a raiz da seção)
- [ ] Estado vazio: "Nenhum curso disponível ainda." quando não há cursos

### AC2 — CursoPage (`/cursos/:moduleId`)
- [ ] **Breadcrumb**: `Cursos › [Título do Curso]` — "Cursos" é link para `/cursos`
- [ ] **Título** do curso (`modules.title`) renderizado como `<h1>`
- [ ] **Subtítulo** (`modules.subtitle`) renderizado abaixo do título — não exibe se nulo/vazio
- [ ] **Botão "Iniciar Curso"** (destaque, cor accent):
  - Leva para a primeira aula (`lessons` com menor `order` que tenha `video_url` ou qualquer tipo)
  - Desabilitado/oculto se não houver aulas cadastradas
  - Texto "Continuar Curso" se o aluno já completou pelo menos 1 aula
- [ ] **Seção "Sobre o Curso"**:
  - Título fixo "Sobre o Curso" em destaque
  - Conteúdo: `modules.description`
  - Seção oculta se `description` for nulo/vazio
- [ ] **Seção "Conteúdo do Curso"**:
  - Título fixo "Conteúdo do Curso"
  - Lista ordenada de aulas (`lessons` por `order ASC`)
  - Cada item: número da aula, título, duração formatada, badge "✓ Concluída" se completa
  - Cada item é clicável → navega para `/cursos/:moduleId/aulas/:lessonId`
  - Se não houver aulas: "As aulas serão adicionadas em breve."
- [ ] Estado de loading enquanto dados carregam
- [ ] Estado de erro se curso não encontrado (404-like)

### AC3 — LessonPage — rotas e breadcrumb
- [ ] Rota atualizada para `/cursos/:moduleId/aulas/:lessonId`
- [ ] Breadcrumb: `Cursos › [Título do Curso] › [Título da Aula]`
  - "Cursos" → link `/cursos`
  - "[Título do Curso]" → link `/cursos/:moduleId`
- [ ] `useParams` usa `moduleId` e `lessonId` (sem alteração na lógica interna)

### AC4 — Rotas e redirecionamentos (`plataforma-main.jsx`)
- [ ] Rota `/cursos` → `CursosPage` (nova)
- [ ] Rota `/cursos/:moduleId` → `CursoPage` (nova)
- [ ] Rota `/cursos/:moduleId/aulas/:lessonId` → `LessonPage` (rota atualizada)
- [ ] Rota `/modulos` → `<Navigate to="/cursos" replace />`
- [ ] Rota `/modulos/:moduleId/aulas/:lessonId` → `<Navigate to="/cursos/:moduleId/aulas/:lessonId" replace />` (usando `useParams` ou componente redirect)
- [ ] Sidebar `Layout.jsx`: link "Cursos & Aulas" aponta para `/cursos`

### AC5 — UX e estilo
- [ ] Inline styles com `var(--token)` — sem CSS externo
- [ ] Cards em `CursosPage` respondem a hover (background sutil)
- [ ] Botão "Iniciar Curso" é visualmente destacado (accent background, bold)
- [ ] Listas de aulas na `CursoPage` usam hover states
- [ ] Nenhum `alert()` ou `confirm()` nativo

### Queries necessárias
```javascript
// CursosPage — lista todos os cursos com contagem de aulas
supabase
  .from('modules')
  .select('id, title, subtitle, order, lessons(id)')
  .order('order', { ascending: true })

// CursoPage — curso + aulas ordenadas
supabase
  .from('modules')
  .select('id, title, subtitle, description, lessons(id, title, video_url, duration, lesson_type, order)')
  .eq('id', moduleId)
  .order('order', { referencedTable: 'lessons', ascending: true })
  .single()
```
