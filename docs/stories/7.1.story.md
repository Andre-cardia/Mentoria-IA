# Story 7.1 — Materiais: DB module_id

## Status

Ready

## Story

**Como** admin,
**Quero** que cada material possa ser associado a um curso no banco de dados,
**Para que** as stories seguintes possam filtrar e agrupar materiais por curso.

## Contexto

Primeira story do Epic 7. Prepara o banco adicionando a FK opcional `module_id` na tabela `materials`. Nenhuma interface é alterada — apenas a fundação de dados.

**Tabela alvo:** `public.materials`
**Campo novo:** `module_id uuid REFERENCES modules(id) ON DELETE SET NULL` (nullable)

Quando o admin excluir um curso, os materiais vinculados não são excluídos — `module_id` passa a `null` e o material vira "Conteúdo Extra" automaticamente.
